To use...

1. prepare the test_seating.txt file
   (can use the seating file produced by Rainbow Grades)

2. if the students have uploaded crib sheets, put them in
   students_notes directory

3. prepare your test template.  Leave space in the upper half of the
   page for the preprinted zone/name/qr code.

4. run:
   python3 make_all.py

5. files will be produced in to_print

6. tweak overlay.tex if desired and re-run
